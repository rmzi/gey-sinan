## Key Findings

### Language Overview
- **Speakers**: ~25,000 (concentrated in Harar, Ethiopia + diaspora)
- **Classification**: Ethio-Semitic (South Semitic branch of Afroasiatic)
- **Writing Systems**: 3 scripts (Arabic, Ethiopic, Latin)
- **Status**: Vulnerable/at-risk language

### Existing Resources
1. **LearnHarariFull** - iOS app with 200+ flashcards (Gangora Learning Group)
2. **Harfi Kitab** - Alphabet book from Australian Harari community (PDF available)
3. **Wolf Leslau's Dictionary** - 2,200 roots with etymologies (PDF available)
4. **harar.city** - Primary online hub for language materials
5. **Glosbe** - Online dictionary with translations

### Major Gaps (Opportunities for Your App)
- No Duolingo-style interactive course
- No structured video lessons
- No comprehensive pronunciation training
- No spaced repetition vocabulary system
- No conversation practice platform
- No modern mobile-first learning experience

## Australian Harari Community Book

The **"Harfi Kitab"** (Harari Alphabet Book) was created by the Australian Saay Harari Association in Melbourne:
- First published: March 2003
- Updated: February 2010
- Won 2009 Arts Portfolio Leadership Awards for orthographic mapping
- PDF: https://harar.city/Content/assets/pdf/Harfi_Kitab.pdf
