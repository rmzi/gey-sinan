# Harari Language: Linguistic Overview

## Basic Information

- **Name**: Harari (also: Adare, Adarinnya, Adere, Aderinya, Gey Sinan, Hararri)
- **Native name**: Gēy Sinan or Gēy Ritma ("language of the City")
- **Speakers**: ~25,000-25,810 (2007 Ethiopian census)
- **Location**: Harari region in southeastern Ethiopia, particularly the city of Harar
- **ISO 639-3**: har

## Linguistic Classification

Harari belongs to the **Afroasiatic language family**:

```
Afroasiatic
└── Semitic
    └── South Semitic
        └── Ethio-Semitic (Ethiopian Semitic)
            └── South Ethio-Semitic
                └── Harari
```

### Relationship to Other Languages

- **Closest relatives**: Eastern Gurage languages (Wolane, Selti-Ulbarag, Zway), Siltʼe
- **Related to**: Amharic, Argobba, Tigrinya (all Ethio-Semitic)
- **Historical connection**: Believed to be linked to the now-extinct Harla language
- **Substratum influence**: According to Wolf Leslau, Sidama (a Cushitic language) is the substratum language and influenced the vocabulary greatly

## Phonology

### Vowel System

Harari has **11 vowel phonemes**:

| Short Vowels | Long Vowels |
|--------------|-------------|
| /i/ | /iː/ |
| /e/ | /eː/ |
| /u/ | /uː/ |
| /o/ | /oː/ |
| /a/ | /aː/ |
| /ä/ (schwa-like, unstressed) | - |

**Vowel Harmony**: Harari exhibits vowel harmony involving assimilation of front/back features, particularly in suffixes. This is a trait shared with neighboring Gurage languages.

### Consonant System

- **24 basic consonants** plus **5 ejective consonants**
- Ejectives represented as: p', t', č', k', q' (glottalized sounds)
- Features typical of Ethiopian Semitic languages

### Phonological Adaptations

When written in Arabic script, modifications were made to accommodate:
- Ejective consonants
- Palatal sounds
- Velar sounds not present in standard Arabic

## Grammar

### Nouns
- Two grammatical numbers: singular and plural
- Broken plurals (common to Ethiopian Semitic languages)

### Morphology
- Agglutinative features
- Verb-final word order (SOV)
- Complex verbal morphology typical of Semitic languages

## Writing Systems

Harari uses **three writing systems**:

### 1. Arabic Script (Modified)
- Original writing system
- 36 letters (28 Arabic + 8 additional for Harari-specific sounds)
- Historically used for Islamic literature and manuscripts

### 2. Ethiopic Script (Ge'ez)
- Adopted by Harari Regional Government in 1999
- Used in official contexts within Ethiopia

### 3. Latin Script
- Developed by Harari Language School (Melbourne) in 1992
- Popular among diaspora communities
- Easier for children to learn
- Uses special characters: ȧ, dh, ḋ, ḣ, gh, ṡ, ṫ, ż (for Arabic loanwords)

### Historical Script
- **Native Harari secret script** - used until the late 1970s

## Endangerment Status

Harari is considered a **vulnerable/at-risk language**:
- Small speaker population (~25,000)
- Diaspora dispersion has fragmented the community
- Language transmission to younger generations is declining
